mac sublime text3 环境设置

Posted on 2020-11-25

  1. 下载后安装

  2. 安装package control

  3. package control->pip->soda 安装soda主题

  1. python2环境 tools->build system-> new build system

    ​ “/usr/bin/python” 为机器上的python2命令所在目录

    ​ 保存为python2.sublime-build

  2. python3环境：与python2类似

  3. golang环境

    ​ package control->安装Golang build
    ​ perferences -> package settings -> Golang Config -> Settings-User
    ​ 配置PATH和GOPATH

    ​ 跳转到package目录下，git clone https://margo.sh/GoSublime
    ​ perferences -> package settings -> GoSublime-> Settings-User

    ​ tools->build system->选择Go
    ​ 编写一个go程序，通过command + shift + b或者 command + b即可以在控制台运行该程序。

  4. 配置

  1. 插件
    soda主题
    BracketHighlighter 括号高亮

    prettyJson json格式化插件

    ConvertToUTF8：将除UTF8编码之外的其他编码文件在 Sublime Text 中转换成UTF8编码
